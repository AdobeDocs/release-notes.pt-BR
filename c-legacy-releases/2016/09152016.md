### Serviço da Experience Cloud ID {#mcvid}

Esta versão aprimora o código do serviço de ID para a versão 1.8.0. Inclui recursos, correções de bugs e outras alterações descritas a seguir.

**Correções e melhorias**

Adicionado ` disableThirdPartyCalls` como um sinalizador booleano opcional, que pode ser definido na função ` Visitor.getInstance`. Quando ` disableThirdPartyCalls= true`, o serviço de ID não fará chamadas para outros domínios. Por padrão, ` disableThirdPartyCalls= false` Consulte [disableThirdPartyCalls](https://marketing.adobe.com/resources/help/en_US/mcvid/?f=mcvid-disablethirdpartycalls.html). (MCID-51)
